# Presence Framework

**Presence** is a forkable personal platform: pages on the outside, `/api/v1` + MCP underneath, and a compiled wiki for search and chat.

## What the sample showcases

- Folder entries with colocated embeds
- KaTeX, tables, and project link fields
- Switchable themes and a floating agent that can navigate the site

See also: [[alex-rivera]]

A future hosted product can swap filesystem publishing adapters for a CMS — see `docs/paid-platform.md`.
